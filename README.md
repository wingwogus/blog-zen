# 🚀 블로그 초안 생성기 (Blog Draft Generator)

사용자의 API Key를 활용하여 비용 부담 없이 고품질 블로그 초안을 생성하는 서비스입니다.

## ✨ 핵심 기능
- **BYOK (Bring Your Own Key)**: 자신의 OpenAI 또는 Gemini API Key를 사용하여 보안과 비용 효율성 확보.
- **모던 UI**: Next.js 16, React 19, Tailwind CSS 4 기반의 세련된 디자인.
- **다크 모드 지원**: 시스템 설정에 맞춘 최적화된 테마 제공.
- **서버 액션 기반**: 안정적이고 빠른 데이터 처리.

## 🛠 기술 스택
- **Framework**: Next.js 16 (App Router)
- **Library**: React 19, Lucide React
- **Styling**: Tailwind CSS 4
- **Language**: TypeScript

## 🚀 시작하기

1. **저장소 클론**
   ```bash
   git clone <repository-url>
   cd blog-draft-gen
   ```

2. **패키지 설치**
   ```bash
   npm install
   ```

3. **개발 서버 실행**
   ```bash
   npm run dev
   ```

4. **API 설정**
   - 웹 접속 후 우측 상단의 설정(⚙️) 아이콘을 클릭합니다.
   - OpenAI 또는 Gemini API Key를 입력하고 저장합니다. (키는 브라우저에만 저장됩니다.)
